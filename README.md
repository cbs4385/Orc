  # Defender of the Orcish Marches

  A tower-defense game where you command the defense of a frontier stronghold under siege. Waves of enemies assault your
   fortress — hold the walls, command defenders, operate the ballista, and keep the tower standing.

  ## Downloads

  | Platform | File | Notes |
  |----------|------|-------|
  | Windows | `DefenderOfTheOrcishMarches-Windows.zip` | 64-bit, Windows 10+ |
  | macOS | `DefenderOfTheOrcishMarches-macOS.zip` | Universal (Intel + Apple Silicon), macOS 12+ |

  ## Installation

  ### Windows

  1. Download `DefenderOfTheOrcishMarches-Windows.zip`
  2. Extract the zip to any folder (e.g. `C:\Games\DefenderOfTheOrcishMarches\`)
  3. Run `Defender of the Orcish Marches.exe`

  > **Windows SmartScreen:** You may see a "Windows protected your PC" warning. Click **More info** then **Run anyway**.
   The game is unsigned but safe.

  ### macOS

  1. Download `DefenderOfTheOrcishMarches-macOS.zip`
  2. Extract the zip (double-click in Finder)
  3. Right-click `Defender of the Orcish Marches.app` and select **Open**

  > **Gatekeeper:** On first launch, macOS will warn the app is from an unidentified developer. Right-click > Open
  bypasses this. You only need to do this once.
  >
  > If you see "the application is damaged", open Terminal and run:
  > ```
  > xattr -cr "/path/to/Defender of the Orcish Marches.app"
  > ```
  > Then right-click > Open again.

  ## Controls

  | Input | Action |
  |-------|--------|
  | **Left Click** | Fire ballista |
  | **Right Click** | Send menial to collect loot |
  | **Scroll Wheel** | Zoom in/out |
  | **U** | Toggle upgrade panel |
  | **1–9** | Buy upgrade (when panel open) |
  | **Tab** | Cycle between ballistas |
  | **Space** | Pause/unpause |
  | **Escape** | Pause menu |

  ### Wall Placement (from upgrade panel)

  | Input | Action |
  |-------|--------|
  | **Left Click** | Place wall |
  | **A / D** | Rotate wall |
  | **Right Click / Escape** | Cancel placement |

  ## System Requirements

  - **OS:** Windows 10+ (64-bit) or macOS 12+
  - **GPU:** Any GPU with DirectX 11 / Metal support
  - **RAM:** 4 GB
  - **Storage:** ~200 MB
